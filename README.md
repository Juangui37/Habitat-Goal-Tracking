# Life Dashboard
 
A personal goal tracking, habit management, and analytics dashboard built with React + Vite.
 
## Features
- SMART Goal builder with AI Coach
- Daily habit tracker with streak tracking
- Analytics page with AI-generated insights
- AI Habit Suggester — auto-suggests habits when a new goal is created
- AI SMART Goal Coach
 
## Dev Setup
```bash
npm install
npm run dev
```
 
## Deploy
Connected to Vercel — pushes to `main` branch auto-deploy.
